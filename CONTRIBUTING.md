# Contributing to Super Memory

Thanks for your interest in contributing!

## What we need most
- Memory backend adapters (Qdrant, pgvector, Weaviate)
- Research goal templates for different professions
- OpenClaw plugin improvements
- Windows/macOS/Linux install hardening
- Documentation and tutorials

## How to contribute
1. Fork the repo
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test: `python automation/smoke_test.py`
5. Submit a pull request

## Security
Found a vulnerability? Please email hello@withmnemo.com before opening a public issue.
Review `automation/memory_sanitizer.py` for the current security model.

## Questions
Open an issue or start a discussion. We respond fast.
